## Installation

1. Clone the repository:
   ```
   git clone git@github.com:acheronx0577/Full-Stack-File-Metadata-Microservice.git
   ```
   ```
   cd Full-Stack-File-Metadata-Microservice
   ```
2. Go to html and change this path of style.css
  ```
      <link href="/styles.css" rel="stylesheet" type="text/css">
  ```
To this and same goes to script.js file too:
  ```
      <link href="./public/styles.css" rel="stylesheet" type="text/css">
  ```
4. Install dependencies:
   ```
   npm install
   ```
   ```
   npm init
   ```
   ```
   npm init -y
   ```
5. Start in http://localhost:3000
   ```
   npm start
   ```
